HabitRPG Calendar

HabitRPG Calendar is a tool that takes your ToDo's from HabitRPG and displays them in calendar format.
The first thing you should do is go to the settings page and put in your HabitRPG user token and key.
This was created for personal use, so it's probably buggy insecure. Use at your own risk.
